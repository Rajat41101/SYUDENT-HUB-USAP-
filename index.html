Access Control

Three roles. Zero confusion.

Clear permission tiers so everyone knows exactly what they can do.

👨‍🎓

General Student

Default role for all registered users. Full read access to all approved materials.

  • ✓ View class details
  • ✓ Download PDF books
  • ✓ Browse all study materials
  • ✗ Upload notes
  • ✗ Manage users
✍️

Verified Contributor

Granted to students with exceptional handwriting and consistently accurate notes.

  • ✓ All student permissions
  • ✓ Upload peer-reviewed notes
  • ✓ Verified badge on profile
  • ✗ Approve / reject content
  • ✗ Manage users
🛠️

Admin

Full platform control. Assigns roles, approves content, and manages all users.

  • ✓ All permissions
  • ✓ Assign contributor role
  • ✓ Approve / reject uploads
  • ✓ Manage all users
  • ✓ Full analytics access
